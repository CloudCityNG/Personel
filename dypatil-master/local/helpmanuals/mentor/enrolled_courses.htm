Approve Enrolled Courses will have tabs for pending approvals and completed approvals.

 

You are here: My Tasks > Course Registration > Approve Enrolled Courses

 

1.Pending Class Approvals

 

Click on 'Pending Class Approvals' tab to access the page.

The page displays the list of enrolled courses. The user can approve or reject the course requests.

 

View Screenshot

 

The user can -

1.Apply filters to manage the view

2.Navigate between the pages

3.Click on symbol to approve or symbol to reject the request.

 

 

 

 

2.Completed Approvals

 

This page allows the user to view the list of completed course approvals. The user can -
