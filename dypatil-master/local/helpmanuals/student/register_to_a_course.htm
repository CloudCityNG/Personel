Register to a course has the following 2 tabs:

 

Course Registration

 

Course Registration page shows the list of courses in a semester.

1.The user can manage the list view by applying available filters

2.Click on 'View Classes' will show the details of class schedule

 

View Screenshot

 

 

 

Note: Student can enroll to a course by clicking on 'View Classes' (if not enrolled to the course)

 

 

Course Approval Status

 

Click on 'Course Approval Status' tab to access the page.

 

This page show the list of course registration requests that are approved.

 

The student has to select the semester in order to view the courses approval status, for a particular semester.
