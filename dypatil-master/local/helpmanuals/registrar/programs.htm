Program is an academic qualification. Schools offer various graduate and under-graduate programs.

 

You are here: University Structure > Programs

 

Click on 'Programs' as shown in the image below or navigate to through the left menu.

 

 

View Programs

 

The following page will be displayed when you click on 'View Programs' tab. Click on symbol to delete the program, symbol to edit the program or symbol to inactivate the program.

 

 

 

Create Program

 

The following page will be displayed when clicked on 'Create Program' tab.

 

 

To create a program -

1.Select School

2.Enter Program name

3.Enter short name for the program

4.Select Program type

5.Select Program level

6.Enter program duration

7.Enter description

8.Click on 'Create Program' button

 

Upload Programs

 

To add programs in bulk, click on 'Upload Programs' tab.

 

 

To upload the departments -

•Click on Choose a file button and select the file. You can even drag and drop the files.

•Select CSV delimiter

•Select Encoding

•Select preview rows

•Click on upload button

 

If your upload is successful, you will view them under 'View Programs' tab.

 

 

Reports

 

You can view the program reports by clicking on 'Reports' tab. Program report displays the statistical data on the number of active and inactive students for each program.
