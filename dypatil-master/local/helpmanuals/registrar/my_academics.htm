My Academics has the following tabs:

 

My Plan

 

My Plan page shows the curriculum information based on the program, that the student has chosen.

 

To view the list of courses that are included in the curriculum, the student has to click on the curriculum name.

 

The user can also download the transcript when the curriculum status will be complete.

Current Classes

 

This page allows the user to view the list of on-going clclasses, for a semester. The list displays course name, class details and other related information.

 

View Screenshot

 

 

 

 

Scheduled Exams

 

The list of exams that are scheduled for a semester can be accessed here. Click on 'Scheduled Exams' tab to access the page.

 

The user can -

1.Apply the filters to manage the view

2.Navigate between the pages

 

View Screenshot

 

 

 

 

My Transcripts

 

Click on 'My Transcripts' tab to access the page.

 

To view the scores for a semester, select semester from the drop down. The list displays the marks scored for each course and the GPA for that semester.

 

View Screenshot

 

 

 

 

Academic Calendar

 

Academic Calendar shows the list of events during the academic term. To access the page, the user has to click on 'Academic Calendar' tab.

 

1.Change academic calendar view (week, month, year)

2.Apply filters to manage the view

3.Download academic calendar

4.Navigate between pages

Evaluations

 

The Student can view the evaluations here. The user can apply filters to manage the view of evaluations and can navigate between pages.
