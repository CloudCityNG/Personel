Classes Management

Classes are created for courses. Classes are scheduled based on the course requirement, number of students, instructors availability.

 

You are here: University Structure > Classes Management

 

To access the page, click on 'Classes Management' as shown in the image below or navigate from the University Structure block.

 

 

View Classes

 

When clicked on 'Classes Management' menu, you will be directed to 'View Classes' tab.

 

 

The above page will be displayed where you can -

 

1.Schedule Class - This can be done either while creating a class or after creating the class. If the registrar has not scheduled the class while creating, then the 'Schedule Class' link will appear (as shown in the image)

 

2.Actions - View clclasses allow the user to perform the following actions

•Delete the class ()

•Edit the class ()

•Inactivate the class ()

•Create Evaluation ()

•Set criteria (completion) for the class ()

•Assign instructor to the class ()

 

 

Create Class

 

To create a new class -

•Select School

•Select the semester

•Select department

•Select course

•Enter Class name and short name, description (if required)

•Enter limit (class limit)

•Select course mode

•Select class type

•Assign instructor by selecting department

•Schedule class

•Click on 'Update Class' button
