Registrars approve/reject an applicant as part of admission process.

 

You are here: University Structure > Admissions

 

Registrar can view the following tabs when clicked on Admissions.

 

New Applicants

 

New Applicants page allows the user to view the list of applicants applied for various programs.

 

1.Filters: Apply filters to manage the view of applicants based on program type, school, program

 

2.The registrar can

•View Files - View the documents submitted by the applicant

•Accept - To approve admission of the applicant

•Reject - To reject admission

•Contact - To contact the applicant through email

 

3.Select - To accept multiple applicants, select the applicants and click on 'Accept' button.

Applicant Report

 

To access applicant report click on 'Applicant Report' tab.

The Applicant report displays application status information of each applicant applied to various programs. The registrar can also edit the report of an applicant by clicking on edit () symbol.

 

1.Apply various filters to manage the view of report.

2.Navigate between pages

Add New Applicant

 

The registrar can add an applicant through this page. The registrar has to select details such as the program type, admission type, student type, program name for the new applicant, enter the fields and then click on ‘submit’ button to add a new applicant.

 

The added applicant will be listed and displayed under 'New Applicants' tab where the approval for admission can be done.

 

 

Add Student

 

This page is to add approved applicants to a program. The registrar has to select the program, student type, enter the fields based on the details furnished by the student.

 

Upload Applicants

 

'Upload Applicants' tab facilitates to add applicants in bulk.

Click on 'Upload Applicants' tab. This will display the following page where you can choose the file (which is in the given format) and click on 'Upload' button
