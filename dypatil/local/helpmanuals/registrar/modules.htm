Modules is a set of courses assigned to a curriculum.

 

You are here: University Structure > Modules

 

Click on 'Modules' as shown in the image below or navigate from the 'University Structure' block on the left menu, to access.

 

 

View Modules

 

Under 'View Modules' tab you can view the list of all the modules created. Here you can manage the modules i.e., you can delete/edit/inactivate a module by clicking on the symbols (delete) or (edit) or (inactivate).

 

 

Assign Courses

 

Once you create a Module, you have to assign courses to it. To assign courses, click on 'Assign Courses' link. The following page will be displayed.

 

To assign courses to a module -

•Select School

•Select Department

•Select courses and

•Click on Assign Courses button

 

 

 

 

 

Create Modules

 

Click on 'Create Module' tab, this will display the following page.

 

 

To create module -

•Select School

•Select Program

•Enter Module name

•Enter description for the module, if required

•Click on Create Module button
