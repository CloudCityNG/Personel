Grade Letters

 Top  Previous  Next

The registrar can define grade letters for grade points here. Under Grade Letters, the registrar has access to the following:

 

You are here: University Structure > Grades > Grade Letters

 

View Grade Letters

 

Under 'View Grade Letters' tab the registrar can -

1.Apply the 'Schools' filter to manage the view based on schools.

2.Navigate between pages

3.Delete () or Edit () or Inactivate () the grade letters.

Create Grade Letter

 

Click on 'Create Grade Letter' tab, to create a new grade letter.

 

1.Select School

2.Enter Letter Grade

3.Enter Mark from value (minimum mark for the grade letter)

4.Enter Mark to value (Maximum mark for the grade letter)

5.Enter the Grade Point based on the given marks range

6.Click on Create Grade Letter button
