Academic Calendar

 Top  Previous  Next

Academic Calendar is the annual publication by the colleges and has the list of events.

 

You are here: University Structure > Academic Calendar

 

The Registrar has access to the following:

 

View Events

 

When clicked on Academic Calendar, you will directed to the following page.

 

 

1.Change view (week, month, year) of academic calendar

2.Apply filters to manage the view

3.Navigate between pages

4.Perform actions: delete an event, edit an event, inactivate an event

 

 

Create Event

 

•To create an event, click on 'Create Event' tab. Fill in all the fields and click on 'Create Event' button.

 

•Events can be created at various levels (Global, School, Program or Semester)

Manage Event Types

 

Events are categorized into types such as general, academic, add & drop, etc.

 

'Manage Event Types' page allows the view the event types created and also to create a new event type.
