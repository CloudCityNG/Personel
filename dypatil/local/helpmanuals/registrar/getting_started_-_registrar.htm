The Getting Started - Registrar guide will help you to know the Registrar's activities and the CobaltLMS navigation, when the user logs in as registrar.

 

Schools

A School is defined as an institute that offers programs. CobaltLMS offers to manage multiple schools.

 

Departments

Schools have various departments. The registrar can assign a department to multiple schools.

 

Programs

Program is an academic qualification. Schools offer various graduate and under-graduate programs.

 

Semesters

The academic year of each program is divided into semesters. Each academic year may have 2 or 3 semester.

 

Curriculums

Each program has a different curriculum which includes a set of courses.

 

Modules

Modules is a set of courses assigned to a curriculum.

 

Courses

Cobalt courses are the units of learning which are created for departments under a school. Courses are assigned to programs in the form Curriculum.

 

Classes Management

Classes are created for courses. Classes are scheduled based on the course requirement, number of students, instructors availability.

 

Academic Calendar

Academic Calendar is the annual publication by the colleges and has the list of events.

 

Admissions

Registrars approve/reject an applicant as part of admission process.

 

Course Registration

Course Registration allows the registrar to approve/reject the requests for enrolled courses and add/drop courses made by the applicants.

 

Evaluations

Registrar can determine the criteria to evaluate students and instructors through evaluations.

 

Grades

To evaluate student's performance. Grades has the following submenu.

•Grade Letters

•Exam Type

•Lecture Type

•Grade Submission

 

Examinations

Examinations can be created here. Through examinations the students will evaluated.

 

Graduation

The students who satisfy all the academic conditions will be awarded degree.

 

Resource Management

Schools have resources such as buildings, floors, classrooms, classroom resources. 'Resource Management' allows to manage these resources of a school.
