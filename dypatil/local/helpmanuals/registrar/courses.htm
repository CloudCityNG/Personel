Courses

Cobalt courses are the units of learning which are created for departments under a school. Courses are assigned to programs in the form Curriculum.

 

You are here: University Structure > Courses

 

Navigate to courses page by clicking on 'Courses' as shown in the image below or through the left menu under 'University Structure' block.

 

 

View Courses

 

The following page will be displayed when you click on 'View Courses' tab. Here you can view the list of courses. Click on symbol to delete the course or symbol to edit the course or symbol to inactivate the course.

 

You can also download the courses by simply clicking on 'download button as shown below.

 

 

 

Create Course

 

Click on 'Create Course' tab will display the following page.

 

 

To create a new course -

•Select School

•Select Department

•Enter Course name

•Enter Course Id

•Enter description, if required

•Select Course type (general or elective)

•Enter credit hours

•Enter course cost

•Click on Create Course button

 

 

Prerequisite

 

Course Prerequisites are the essential course criteria defined for a course.

 

Click on 'Prerequisite' tab to access the page. The following page will be displayed.

 

 

To add a prerequisite for a course -

•Select the School

•Select the department

•Select the course (for which you are setting the prerequisites)

•Select the department (from which you want to select the prerequisite course)

•Select the prerequisite course

•Click on Prerequisite Course button

 

The defined prerequisite courses will be displayed in the list at the bottom of the page. You can also unassign the prerequisite courses by clicking on .

 

 

Equivalent

 

Course Equivalents are the comparable courses that the students take up from other colleges/institutes.

 

Click on 'Equivalent' tab to define equivalent course(s).

 

 

To add an equivalent course -

•Select the School

•Select department

•Select course (for which you are adding an equivalent)

•Select department

•Select equivalent course

•Click on 'Equivalent Course' button.

 

 

Upload Courses

 

In order to upload courses in bulk, click on 'Upload Courses' tab.

Reports

 

To view the course report, click on 'Reports' tab.

 

Course report displays the information about number of enrollments, department, assigned programs.

 

You can also apply filters to manage the view based on departments, search for a course, navigate to different pages.
